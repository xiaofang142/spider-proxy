## 核心交互

### 2.1 一键开始/暂停/停止

**状态机：**
```
已停止 ──[开始]──► 运行中
   ▲                  │
   │                  │
   └────[停止]────────┘
   
运行中 ──[暂停]──► 已暂停
   ▲                  │
   │                  │
   └────[继续]────────┘
```

**交互细节：**
- 按钮状态随抓包状态变化
- 状态变更有确认对话框 (停止时)
- 变更后有 Toast 提示

**动画：**
- 按钮图标切换：300ms 缩放淡入淡出
- 状态指示器：200ms 颜色过渡

---

### 2.2 抓包列表搜索

**搜索模式：**
```
普通搜索：输入关键词 → 匹配 URL/域名
自然语言：输入 "GET 请求 200 状态" → 解析条件
高级搜索：点击过滤器图标 → 打开条件面板
```

**交互细节：**
- 输入时实时过滤 (300ms 防抖)
- 支持关键词高亮
- 搜索结果计数显示
- 清空按钮：输入后出现

**自然语言解析示例：**
| 输入 | 解析结果 |
|------|----------|
| `GET` | method = GET |
| `200` | status = 200 |
| `api.example.com` | host contains "api.example.com" |
| `GET 200 api` | method = GET AND status = 200 AND url contains "api" |

---

### 2.3 快速过滤芯片

**交互状态：**
```
默认状态：
┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐
│全部 │ │HTTP │ │HTTPS│ │ WS  │
└─────┘ └─────┘ └─────┘ └─────┘

选中状态 (点击"HTTP")：
┌─────┐ ┌─────────┐ ┌─────┐ ┌─────┐
│全部 │ │× HTTP   │ │HTTPS│ │ WS  │
└─────┘ └─────────┘ └─────┘ └─────┘
```

**交互细节：**
- 点击芯片：切换选中状态
- 点击 ×：移除该过滤条件
- 多选支持：可同时选择多个协议
- 点击"全部"：清空其他选择

---

- 内容懒加载：首次点击时加载

### 2.6 Body 查看器格式切换

**格式选项：**
```
Raw ──► 原始文本
Preview ──► 渲染预览 (HTML/图片)
JSON ──► 格式化 JSON (可折叠)
Hex ──► 十六进制视图
```

**交互细节：**
- 下拉选择切换格式
- JSON 视图支持节点折叠/展开
- 支持一键复制
- 支持搜索内容

### 2.7 批量操作

**操作模式：**
```
1. 长按列表项 → 进入选择模式
2. 点击其他项 → 多选
3. 底部出现操作栏
4. 选择操作 → 批量执行
```

**批量操作选项：**
- 导出选中
- 删除选中
- 标记选中
- 复制到剪贴板

**交互细节：**
- 选择模式：项左侧出现复选框
- 全选按钮：操作栏左侧
- 计数显示：已选择 X 项
- 取消：点击空白处或 ESC

## 无障碍交互

### 5.1 键盘导航
- Tab 顺序：从上到下，从左到右
- 焦点可见：2px Spider Orange 边框
- 跳过链接：页面顶部"跳到主内容"

### 5.2 屏幕阅读器
- 所有图标按钮有 aria-label
- 状态变化有 live region 通知
- 表单有 label 关联
- 错误信息有 aria-describedby

### 5.3 减少动画
- 支持 prefers-reduced-motion
- 动画时长减少 50%
- 移除视差和缩放效果
