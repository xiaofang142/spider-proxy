### Spider Proxy 配色

**主色调：**
```
Spider Orange: #FF6B35
RGB: 255, 107, 53
HSL: 16°, 100%, 60%
```

**生成工具色：**
```bash
# 使用 Python 生成
python3 -c "
import colorsys

# Spider Orange
r, g, b = 255/255, 107/255, 53/255
h, l, s = colorsys.rgb_to_hls(r, g, b)

# 生成深色变体 (暗 10%)
darker = colorsys.hls_to_rgb(h, l*0.9, s)
print(f'Darker: #{int(darker[0]*255):02X}{int(darker[1]*255):02X}{int(darker[2]*255):02X}')

# 生成浅色变体 (亮 10%)
lighter = colorsys.hls_to_rgb(h, min(l*1.1, 1), s)
print(f'Lighter: #{int(lighter[0]*255):02X}{int(lighter[1]*255):02X}{int(lighter[2]*255):02X}')
"
```

### 导出规格

#### SVG 导出 (已生成 ✅)
- 用于开发参考
- 矢量可编辑
- 可在浏览器中直接查看
- 文件位置：`UI 设计稿/*.svg`

#### PNG 导出 (需手动转换)
| 倍率 | 尺寸 | 用途 |
|------|------|------|
| @1x | 1440×900 | 文档展示、网页 |
| @2x | 2880×1800 | Retina 屏幕、打印 |
| @3x | 4320×2700 | 高清展示、印刷 |

**手动转换方法：**
```bash
# 使用 Inkscape (需先安装)
inkscape --export-filename=dashboard@2x.png --export-dpi=144 dashboard.svg

# 或使用在线工具：cloudconvert.com/svg-to-png
```

#### PDF 导出 (需手动转换)
- 用于打印和分享
- 包含所有页面
- 带书签导航

**手动转换方法：**
```bash
# 使用 Inkscape
inkscape --export-filename=designs.pdf *.svg

# 或使用浏览器打印功能：打开 SVG → 打印 → 保存为 PDF
```

## Python SVG 生成脚本

### 依赖安装

```bash
pip install svgwrite pillow
```

### 脚本位置

设计稿生成脚本位于：
```
/Users/xiaofang/documents/www/docker/spider-proxy/design/generate-ui-designs.py
```

### 使用方法

```bash
# 生成所有设计稿
python3 generate-ui-designs.py

# 生成单个页面
python3 generate-ui-designs.py --page dashboard

# 指定输出目录
python3 generate-ui-designs.py --output /path/to/output

# 生成 PNG 和 SVG
python3 generate-ui-designs.py --format svg,png
```
