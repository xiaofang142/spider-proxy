## 一、设计原则

### 1.1 核心设计理念

```
┌─────────────────────────────────────────┐
│                                         │
│   简洁 (Simple)                         │
│   • 减少视觉噪音                        │
│   • 突出核心内容                        │
│   • 一屏一任务                          │
│                                         │
│   高效 (Efficient)                      │
│   • 3 次点击内达核心功能                 │
│   • 支持手势快捷操作                    │
│   • 智能默认值                          │
│                                         │
│   一致 (Consistent)                     │
│   • 统一的交互模式                      │
│   • 一致的视觉语言                      │
│   • 可预测的行为                        │
│                                         │
│   反馈 (Responsive)                     │
│   • 即时操作反馈                        │
│   • 清晰的状态指示                      │
│   • 加载进度可视化                      │
│                                         │
└─────────────────────────────────────────┘
```

### 1.2 设计目标

| 指标 | 目标值 | 测量方式 |
|-----|--------|---------|
| 首次使用学习成本 | < 3 分钟 | 用户测试 |
| 核心功能可达性 | ≤ 3 次点击 | 交互分析 |
| 视觉层次清晰度 | > 4.5/5 | 用户评分 |
| 操作满意度 | > 4.0/5 | NPS 调查 |

## 三、字体系统

### 3.2 字号规范

| 用途 | 字号 | 字重 | 行高 |
|-----|------|------|------|
| 大标题 | 24sp | Bold | 32sp |
| 页面标题 | 20sp | Bold | 28sp |

| 字重 | 用途 |

## 四、间距系统

### 4.1 基础间距单位

```
基础单位：4dp

间距等级：
4dp   - 最小间距 (图标与文字)
8dp   - 小组件内部间距
12dp  - 相关元素间距
16dp  - 标准间距 (卡片内边距)
24dp  - 大间距 (卡片之间)
32dp  - 区域分隔
48dp  - 大区域分隔
```

## 七、动效规范

### 7.1 时长规范

| 动效类型 | 时长 | 缓动 |
|---------|------|------|
| 小动效 (按钮) | 150ms | ease-out |
| 中动效 (卡片) | 250ms | ease-out |
| 大动效 (页面) | 300ms | ease-out |
| 复杂动效 | 500ms | custom |

### 7.2 缓动曲线

```
标准缓动 (ease-out):
╭───────────────────
│              ╱
│           ╱
│        ╱
│     ╱
│  ╱
╰───────────────────

加速缓动 (ease-in):
╭───────────────────
│           ╱
│        ╱
│     ╱
│  ╱
│╱
╰───────────────────
```

### 7.3 页面转场

```
进入页面：从右向左滑入，300ms
离开页面：从右向左滑出，300ms
对话框：从底部滑入 + 淡入，250ms
```

### 7.4 列表项动效

```
列表加载：逐项淡入，每项延迟 50ms
删除：向左滑出 + 缩小，250ms

```

## 九、无障碍设计

### 9.1 对比度要求

| 内容类型 | 最小对比度 |
|---------|-----------|
| 大文字 (18sp+) | 3:1 |
| 普通文字 | 4.5:1 |
| 图标 | 3:1 |

### 9.2 触摸目标

- 最小尺寸：48x48dp
- 推荐尺寸：56x56dp
- 目标间距：≥ 8dp

### 9.3 内容描述

- 所有图标按钮需设置 contentDescription
- 图片需设置 alt 文本
- 表单需设置 label

### 9.4 键盘导航

- 支持 Tab 键导航
- 支持 Enter 键确认
- 支持 Esc 键取消
