## 新增功能模块

### A. 流量分析增强模块

#### 域名/IP 热力图 (DomainHeatmap)
- Top N 域名排行显示
- 进度条可视化 + 百分比标注
- 点击展开域名详情
- 支持过滤和搜索操作

#### 请求时间线 (RequestTimeline)
- Waterfall 瀑布图展示请求生命周期
- 支持 6 阶段展示：DNS/TCP/SSL/发送/等待/接收
- 慢请求自动警告标记
- 支持详细数据模式和估算模式

#### 流量分析页面 (AnalysisPage)
- 整合所有 A 模块功能
- 流量概览卡片 (总请求/总流量/平均耗时/域名数)
- 域名分布热力图
- 流量趋势图表 (上传/下载)
- 异常检测 (流量突增/错误率/慢响应/可疑活动)
- 请求时间线详情

**新增文件**:
- `lib/ui/widgets/domain_heatmap.dart`
- `lib/ui/widgets/request_timeline.dart`
- `lib/ui/pages/analysis_page.dart`
- `lib/features/traffic/traffic_analyzer.dart` (增强)
- `lib/main.dart` (更新导航)

### B. 请求重写模块

#### 请求重写引擎 (RequestRewriter)
- 支持修改请求头 (添加/修改/删除)
- 支持修改响应头
- 支持 URL 重定向
- 基于规则匹配 (域名/URL/方法/Header)
- 支持规则导入导出 (JSON)

#### 重写规则管理页面 (RewriteRulesPage)
- 规则列表展示 (类型图标/条件描述/动作描述)
- 规则启用/禁用开关
- 规则添加/编辑/删除
- 规则测试功能

#### 请求重放器 (RequestReplayer)
- 重新发送已捕获的请求
- 支持设置 Header 和 Body
- 超时控制
- 生成 cURL 命令
- 生成 HTTPie 命令
- 生成 PowerShell 命令

**新增文件**:
- `lib/core/rewriter/request_rewriter.dart`
- `lib/core/rewriter/request_replayer.dart`
- `lib/ui/pages/rewrite_rules_page.dart`
- `lib/ui/pages/settings_page.dart` (更新)

### C. 断点调试模块

#### 断点调试引擎 (BreakpointDebugger)
- 支持请求前断点
- 支持响应前断点
- 断点暂停/继续/取消
- 修改请求/响应后继续
- 规则导入导出 (JSON)

#### 断点调试页面 (BreakpointDebuggerPage)
- 全局断点启用/禁用开关
- 断点规则列表展示
- 当前暂停的请求/响应卡片
- 继续/修改/取消操作按钮
- 规则添加/编辑对话框
- 暂停时长实时显示

**新增文件**:
- `lib/core/debugger/breakpoint_debugger.dart`
- `lib/ui/pages/breakpoint_debugger_page.dart`
- `lib/ui/pages/settings_page.dart` (更新)

---

### D. 脚本系统模块

#### 脚本引擎 (ScriptEngine)
- 支持 JavaScript 脚本执行
- 请求拦截 (onRequest)
- 响应拦截 (onResponse)
- 通用脚本 (universal)
- 脚本启用/禁用控制
- 脚本导入导出 (JSON)

#### 脚本管理页面 (ScriptManagerPage)
- 脚本列表展示 (名称/描述/类型)
- 脚本启用/禁用开关
- 脚本创建/编辑/删除
- 代码编辑器 (monospace 字体)
- 加载内置模板按钮
- 脚本运行/导出操作

#### 内置脚本模板 (BuiltInScripts)
- 添加自定义 Header
- 删除敏感 Header
- 请求日志记录
- Mock 响应
- 修改 User-Agent
- 添加时间戳
- 广告拦截
- 启用 CORS

**新增文件**:
- `lib/core/script/script_engine.dart`
- `lib/ui/pages/script_manager_page.dart`
- `lib/ui/pages/settings_page.dart` (更新)

## 使用指南

### 流量分析
1. 点击底部导航栏「分析」标签
2. 查看流量概览、域名分布、流量趋势
3. 点击域名可查看详情或进行过滤

### 请求重写
1. 进入设置页 → 高级功能 → 请求重写
2. 点击 + 添加规则
3. 设置匹配条件和执行动作
4. 启用规则后生效

### 断点调试
1. 进入设置页 → 高级功能 → 断点调试
2. 添加断点规则 (请求前/响应前)
3. 启用全局断点开关
4. 当请求/响应匹配规则时自动暂停
5. 可继续、修改或取消

### 脚本系统
1. 进入设置页 → 高级功能 → 脚本系统
2. 点击「加载内置模板」或「新建脚本」
3. 编辑脚本代码
4. 启用脚本后自动执行
