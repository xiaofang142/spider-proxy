### 2.4 ScriptEngineV2 模块 (完整 JavaScript 引擎)

**实现文件**:
- `lib/core/script/script_engine_v2.dart` - 引擎核心
- `lib/ui/pages/script_manager_page.dart` - 脚本管理页面 (更新)
- `pubspec.yaml` - 添加 dart_eval 和 flutter_highlight 依赖

**核心功能**:
```dart
class ScriptEngineV2 {
  EvalDart? _vm;
  bool _enabled = true;
  final List<ScriptV2> _scripts = [];

  Future<void> initialize() async {
    _vm = EvalDart();
    await _registerLibraries();
  }

  Future<void> _registerLibraries() async {
    // proxy://context 库 - ScriptContext API
    _vm!.registerLibrary('proxy://context', {...});

    // proxy://utils 库 - 工具函数
    _vm!.registerLibrary('proxy://utils', {
      'print': Func1((msg) => print('[Script] $msg')),
      'jsonDecode': Func1((str) => str), // TODO: 实现
      'jsonEncode': Func1((obj) => obj.toString()), // TODO: 实现
    });
  }

  Future<ScriptContext?> executeOnRequest(...);
  Future<ScriptContext?> executeOnResponse(...);
}
```

**ScriptContext API**:
| 方法 | 说明 |
|------|------|
| `ctx.url` | 获取/设置请求 URL |
| `ctx.method` | 获取/设置 HTTP 方法 |
| `ctx.headers` | 获取请求头 Map |
| `ctx.body` | 获取/设置请求体 |
| `ctx.statusCode` | 获取/设置状态码 |
| `ctx.setHeader(name, value)` | 设置 Header |
| `ctx.removeHeader(name)` | 删除 Header |
| `ctx.setBody(body)` | 设置响应体 |
| `ctx.setUrl(url)` | 设置 URL |
| `ctx.setMethod(method)` | 设置方法 |
| `ctx.abort()` | 中止请求/响应 |

**内置脚本模板** (8 个):
1. `addCustomHeader()` - 添加自定义 Header
2. `removeHeader()` - 删除敏感 Header
3. `logRequests()` - 请求日志
4. `mockResponse()` - Mock 响应
5. `modifyUserAgent()` - 修改 User-Agent
6. `addTimestamp()` - 添加时间戳
7. `blockAds()` - 广告拦截
8. `enableCors()` - 启用 CORS

**脚本示例**:
```javascript
import 'proxy://context' as context;
import 'proxy://utils' as utils;

void onRequest(ctx) {
  ctx.setHeader('X-Custom-Header', 'SpiderProxy-v3');
  ctx.removeHeader('Cookie');

  if (ctx.url.contains('ads.example.com')) {
    ctx.abort();
  }
}

void onResponse(ctx) {
  ctx.setHeader('Access-Control-Allow-Origin', '*');
  utils.print('Response: ${ctx.statusCode}');
}
```

### 3.2 缓存策略 (BodyMapper)

```dart
class BodyCache {
  final Map<String, _CacheEntry> _cache = {};
  static const _ttl = Duration(seconds: 5);

  Future<String?> get(String key) async {
    final entry = _cache[key];
    if (entry == null) return null;
    if (entry.expiresAt.isBefore(DateTime.now())) {
      _cache.remove(key);
      return null;
    }
    return entry.content;
  }

  void set(String key, String content) {
    _cache[key] = _CacheEntry(
      content: content,
      expiresAt: DateTime.now().add(_ttl),
    );
  }
}
```
