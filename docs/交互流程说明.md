## 九、状态定义

### 9.1 应用状态

| 状态 | 说明 | 可执行操作 |
|-----|------|-----------|
| Idle | 空闲，未抓包 | 开始抓包、查看历史 |
| Capturing | 抓包进行中 | 暂停、停止、导出 |
| Paused | 抓包已暂停 | 继续、停止、导出 |
| Stopping | 停止中 | 等待完成 |

### 9.2 证书状态

| 状态 | 说明 | 可执行操作 |
|-----|------|-----------|
| NotInstalled | 未安装 | 安装证书 |
| Installed | 已安装 | 查看、导出、删除 |
| Expired | 已过期 | 重新安装 |
| Invalid | 无效 | 重新安装 |

### 9.3 VPN 状态

| 状态 | 说明 | 可执行操作 |
|-----|------|-----------|
| Disconnected | 未连接 | 连接 |
| Connecting | 连接中 | 取消 |
| Connected | 已连接 | 断开 |
| Error | 错误 | 重试、查看帮助 |
