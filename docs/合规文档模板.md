# Spider Proxy 技术说明文档

## 一、产品概述

Spider Proxy 是一款面向开发者的网络调试工具，用于帮助开发者分析自己开发的应用的网络请求，调试 API 接口，优化网络性能。

## 二、技术原理

### 2.1 抓包原理

Spider Proxy 采用本地 VPN 代理模式实现网络请求捕获：

1. 应用在用户设备上创建本地 VPN 服务
2. 用户将需要调试的应用配置为使用此代理
3. 所有网络请求经过本地代理
4. 代理记录请求和响应的完整信息
5. 数据仅在本地显示和存储，不上传任何服务器

### 2.2 HTTPS 解密原理

HTTPS 解密功能（可选）采用以下流程：

1. 应用生成自签名证书
2. 用户手动安装证书到设备信任列表
3. 代理使用此证书与客户端建立 HTTPS 连接
4. 代理同时与真实服务器建立 HTTPS 连接
5. 数据在本地解密后显示给用户

**安全说明：**
- 证书仅在用户设备上有效
- 证书不会上传到任何服务器
- 用户可随时卸载证书
- 解密的数据仅本地显示

### 2.3 数据流向

```
用户设备 → 本地代理 → 真实服务器
    ↓
本地存储（不上传）
```
