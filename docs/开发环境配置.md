## 二、Flutter 安装

### 2.1 macOS 安装

```bash
# 方法一：使用 Homebrew
brew install --cask flutter

# 方法二：手动下载
cd ~
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:$HOME/flutter/bin"

# 添加到 ~/.zshrc 或 ~/.bashrc
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.zshrc
source ~/.zshrc
```

### 2.2 Windows 安装

```powershell
# 下载 Flutter SDK
# https://storage.googleapis.com/flutter_infra_release/releases/stable/windows/flutter_windows_3.24.0-stable.zip

# 解压到 C:\src\flutter

# 添加环境变量
[System.Environment]::SetEnvironmentVariable('PATH', "$env:PATH;C:\src\flutter\bin", [System.EnvironmentVariableTarget]::User)

# 验证安装
flutter doctor
```

### 2.3 Linux 安装

```bash
# 安装依赖
sudo apt-get install -y curl git unzip xz-utils zip libglu1-mesa

# 下载 Flutter
cd ~
git clone https://github.com/flutter/flutter.git -b stable
export PATH="$PATH:$HOME/flutter/bin"

# 添加到 ~/.bashrc
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.bashrc
source ~/.bashrc
```

## 三、Android 环境配置

### 3.1 安装 Android Studio

```bash
# macOS
brew install --cask android-studio

# Windows
# 下载安装包：https://developer.android.com/studio

# Linux
sudo snap install android-studio --classic
```

### 3.2 配置 Android SDK

1. 打开 Android Studio
2. 进入 `Preferences` > `Appearance & Behavior` > `System Settings` > `Android SDK`
3. 安装以下组件：
   - Android SDK Platform (API 24-35)
   - Android SDK Build-Tools
   - Android Emulator
   - Android SDK Platform-Tools

### 3.3 配置环境变量

```bash
# macOS/Linux
export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/platform-tools

# Windows (PowerShell)
[Environment]::SetEnvironmentVariable('ANDROID_HOME', 'C:\Users\你的用户名\AppData\Local\Android\Sdk', [System.EnvironmentVariableTarget]::User)
```

### 3.4 创建 Android 模拟器

```bash
# 列出可用镜像
avdmanager list target

# 创建 AVD
avdmanager create avd -n pixel_6 -k "system-images;android-34;google_apis_playstore;arm64-v8a" -d pixel_6

# 启动模拟器
emulator -avd pixel_6
```

```bash

# 安装 iOS 模拟器运行时
```

## 六、开发工具配置

### 6.1 VS Code 配置

安装扩展：
- Flutter
- Dart
- Flutter Widget Snippets

**settings.json:**
```json
{
  "dart.lineLength": 80,
  "dart.formatOnSave": true,
  "editor.formatOnSave": true,
  "editor.tabSize": 2,
  "files.trimTrailingWhitespace": true,
  "files.insertFinalNewline": true,
  "[dart]": {
    "editor.defaultFormatter": "Dart-Code.dart-code"
  }
}
```

### 6.2 Android Studio 配置

安装插件：
- Flutter
- Dart

**配置项：**
1. `Preferences` > `Languages & Frameworks` > `Flutter`
   - 设置 Flutter SDK 路径
2. `Preferences` > `Editor` > `Code Style` > `Dart`
   - 启用 Format on save
