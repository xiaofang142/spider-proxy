## 四、风险管理

### 4.1 风险矩阵

```
                    影响程度
              低 ←──────────→ 高
            ┌─────────────────────┐
          高 │ 证书兼容性         │ 性能不达标
      发    │ (中概率，高影响)    │ (低概率，高影响)
      生    ├─────────────────────┤
      概    │ 需求变更           │ HTTPS 固定
      率    │ (中概率，中影响)    │ (高概率，中影响)
          低 │                   │
            └─────────────────────┘
```

### 4.2 应对策略

| 风险 | 应对策略 | 触发条件 |
|-----|---------|---------|
| 证书兼容性 | 提前多机型测试 | 3 款以上机型失败 |
| 性能不达标 | 早期性能基线 | 延迟 > 200ms |
| 需求变更 | 敏捷迭代 | 重大需求变更 |
| HTTPS 固定 | 文档说明 + 社区方案 | 用户反馈集中 |
