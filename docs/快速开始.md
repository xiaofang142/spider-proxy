## 🚀 第一步：创建项目 (5 分钟)

### 1.1 使用 OpenCode CLI 创建项目
```bash
# 进入目标目录
```

## 📦 第三步：配置依赖 (5 分钟)

### 3.1 编辑 pubspec.yaml
```bash
# 使用 OpenCode 添加依赖
opencode run -f pubspec.yaml "添加以下依赖：http: ^1.1.0, provider: ^6.0.5, hive: ^2.2.3, logger: ^2.0.0, fl_chart: ^0.65.0"
```

### 3.2 或手动添加
```yaml
# pubspec.yaml
dependencies:
  flutter:
    sdk: flutter
  
  # 网络
  http: ^1.1.0
  
  # 状态管理
  provider: ^6.0.5
  
  # 本地存储
  hive: ^2.2.3
  hive_flutter: ^1.1.0
  
  # 日志
  logger: ^2.0.0
  
  # 图表
  fl_chart: ^0.65.0

dev_dependencies:
  flutter_test:
    sdk: flutter
  hive_generator: ^2.0.1
  build_runner: ^2.4.0
```

### 3.3 安装依赖
```bash
flutter pub get
```

## 💻 第四步：开发核心功能 (10 分钟)

### 4.1 生成 HTTP 代理服务器
```bash
opencode run "生成 HTTP 代理服务器代码，实现以下功能：
1. 启动/停止代理服务器
2. 监听指定端口
3. 转发 HTTP 请求
4. 记录请求和响应
5. 支持连接池管理

文件位置：lib/core/proxy/http_proxy.dart"
```

### 4.2 生成数据模型
```bash
opencode run "生成 HttpRequest 数据模型，包含字段：
- id: String
- method: String
- url: String
- headers: Map<String, String>
- body: String
- timestamp: DateTime
- responseCode: int
- responseTime: int

支持 JSON 序列化，使用 hive 存储。
文件位置：lib/models/http_request.dart"
```

### 4.3 生成首页
```bash
opencode run "生成首页，包含以下组件：
1. AppBar 显示标题和状态指示器
2. 流量统计图表 (使用 fl_chart)
3. 启动/停止代理按钮
4. 抓包数量显示
5. 底部导航栏 (首页、抓包、设置)

使用 Provider 状态管理。
文件位置：lib/ui/pages/home_page.dart"
```

### Q1: Flutter 命令找不到
```bash

which flutter
flutter doctor

# 如果未安装，访问 https://flutter.dev/docs/get-started/install
```

### Q2: OpenCode CLI 命令找不到
```bash
# 检查安装
which opencode
opencode --version

# 如果未安装
npm install -g opencode-ai
# 或
brew install opencode
```

### Q3: 依赖安装失败
```bash
# 清理缓存
flutter clean
flutter pub cache repair
flutter pub get

# 检查网络
ping pub.dev
```

### Q4: iOS 构建失败
```bash
# 进入 iOS 目录
cd ios
pod install
cd ..

# 清理构建
flutter clean
flutter build ios
```

### Q5: Android 构建失败
```bash
# 检查 Android SDK
flutter doctor --android-licenses

# 清理构建
flutter clean
flutter build apk
```
