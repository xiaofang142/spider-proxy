#### 3.1.2 核心类设计

```dart
/// 请求体替换引擎
class BodyMapper {
  final List<BodyMapperRule> _rules = [];
  final BodyCache _cache = BodyCache();

  /// 添加规则
  void addRule(BodyMapperRule rule);

  /// 处理请求体
  Future<String?> mapRequestBody(
    String url,
    String method,
    String body
  );

  /// 处理响应体
  Future<String?> mapResponseBody(
    String url,
    int statusCode,
    String body
  );
}

/// 替换规则
class BodyMapperRule {
  final String id;
  final String name;
  final bool enabled;
  final BodyMapType type; // local 或 remote
  final MatchCondition condition;
  final String? localPath;  // Map Local 使用
  final String? remoteUrl;  // Map Remote 使用
  final MapType mapType;    // 替换请求体或响应体

  bool matches(String url, String method);
  Future<String?> apply(String originalBody);
}

/// 缓存管理器
class BodyCache {
  final Map<String, _FileCacheEntry> _fileCache = {};
  final Map<String, _ResponseCacheEntry> _responseCache = {};

  // 文件监听实现热更新
  void watchFile(String path);
  void invalidate(String key);
}
```

#### 3.1.3 处理流程

```
请求体替换流程:
1. 请求到达 → 2. 匹配规则 → 3. 读取本地文件 → 4. 替换 Body → 5. 继续发送

响应体替换流程:
1. 响应到达 → 2. 匹配规则 → 3. 读取本地文件/远程响应 → 4. 替换 Body → 5. 返回客户端
```

```dart
/// WebSocket 代理
class WebSocketProxy {
  final WebSocketConnectionManager _connectionManager;
  final ValueNotifier<List<WebSocketConnection>> connections;

  /// 是否支持 WebSocket
  bool get isSupported => true;

  /// 拦截 WebSocket 连接
  Future<WebSocketConnection?> interceptConnection(
    String url,
    Map<String, String> headers,
  );

  /// 处理消息
  void handleMessage(WebSocketConnection connection, dynamic message);

  /// 关闭连接
  void handleClose(WebSocketConnection connection, int? code, String? reason);
}

/// WebSocket 连接
class WebSocketConnection {
  final String id;
  final String url;
  final DateTime connectedAt;
  WebSocketState state;

  // 消息历史
  final List<WebSocketMessage> messages;

  // 元数据
  final Map<String, String> headers;
  final String? protocol;
  final String? extensions;

  Duration get connectionDuration => DateTime.now().difference(connectedAt);
}

/// WebSocket 消息
class WebSocketMessage {
  final String id;
  final String connectionId;
  final DateTime timestamp;
  final MessageDirection direction; // client_to_server 或 server_to_client
  final MessageType type; // text 或 binary
  final dynamic payload;

  int get size => payload is String ? (payload as String).length : (payload as Uint8List).length;
}
```

## 五、关键技术难点与解决方案

### 5.1 Map Local 性能优化

**难点**: 大文件读取和频繁 IO 影响性能

**解决方案**:
```dart
class BodyCache {
  // 1. 文件内容缓存
  final Map<String, _CacheEntry> _cache = {};

  // 2. 文件监听，自动失效
  void watchFile(String path) {
    final watcher = FileWatcher(path);
    watcher.events.listen((event) {
      _cache.remove(path); // 文件变更时失效
    });
  }

  // 3. 懒加载
  Future<String> getFileContent(String path) async {
    if (_cache.containsKey(path) && !_cache[path]!.isExpired) {
      return _cache[path]!.content;
    }
    final content = await File(path).readAsString();
    _cache[path] = _CacheEntry(content, DateTime.now());
    return content;
  }
}
```

### 5.2 WebSocket 长连接管理

**难点**: 多个长连接并发时的资源管理

**解决方案**:
```dart
class WebSocketConnectionManager {
  final Map<String, WebSocketConnection> _connections = {};
  final _maxConnections = 50; // 最大连接数限制

  Future<WebSocketConnection> createConnection(String url) async {
    if (_connections.length >= _maxConnections) {
      // 关闭最早的连接
      final oldest = _connections.values.first;
      await oldest.close();
    }

    final connection = WebSocketConnection(url);
    _connections[connection.id] = connection;
    return connection;
  }

  void removeConnection(String id) {
    _connections.remove(id);
  }
}
```

### 5.3 dart_eval 安全沙箱

**难点**: 防止恶意脚本执行

**解决方案**:
```dart
class ScriptEngineV2 {
  EvalDart _createSandbox() {
    final vm = EvalDart();

    // 1. 限制可访问的库
    vm.registerLibrary('proxy://context', {
      'url': Getter((_) => context.url),
      'method': Getter((_) => context.method),
      'headers': Getter((_) => context.headers),
      'body': Getter((_) => context.body),
      'setHeader': Function2((name, value) => context.setHeader(name, value)),
      'setBody': Function1((body) => context.setBody(body)),
      'abort': Function0(() => context.abort()),
    });

    // 2. 禁止 IO 操作
    // 3. 禁止网络请求
    // 4. 执行超时限制
    return vm;
  }
}
```

## 七、兼容性设计

### 7.1 脚本格式兼容

v2.0 使用简化的 `@marker`语法，v3.0 支持完整 JavaScript:

```dart
// v2.0 简化语法
// @modify-header: X-Custom=Value

// v3.0 完整语法
function onRequest(context) {
  context.setHeader('X-Custom', 'Value');
  return context;
}

// 兼容层：自动转换
class ScriptCompatibility {
  Script convertV2ToV3(ScriptV2 v2Script) {
    // 将 @marker 语法转换为完整 JS
    return ScriptV3(...);
  }
}
```
