## 四、P0 核心功能需求（v1.0）

### 4.1 核心引擎

#### F-001: HTTP 代理服务器

**描述**: 实现本地 HTTP 代理服务器，接收并转发 HTTP 请求

**功能要求**:
- 监听本地端口（默认 8888）
- 接收 HTTP 请求并解析
- 转发请求到目标服务器
- 接收响应并返回给客户端
- 记录请求/响应数据

**验收标准**:
- [ ] 支持 HTTP 1.1 协议
- [ ] 支持 GET/POST/PUT/DELETE 等方法
- [ ] 请求转发延迟 < 50ms
- [ ] 支持并发连接数 > 100

---

#### F-002: HTTPS MITM 解密

**描述**: 实现中间人代理，解密 HTTPS 流量

**功能要求**:
- 拦截 HTTPS CONNECT 请求
- 动态生成主机证书
- 与客户端建立 SSL 连接
- 与目标服务器建立 SSL 连接
- 解密并记录流量

**验收标准**:
- [ ] 支持 TLS 1.2/1.3
- [ ] 证书动态生成时间 < 100ms
- [ ] 支持主流加密套件
- [ ] 解密成功率 > 95%

---

#### F-003: CA 证书管理

**描述**: 生成和管理 CA 根证书

**功能要求**:
- 生成自签名 CA 证书
- 私钥安全存储（Android KeyStore）
- 证书 24 小时自动过期
- 支持证书导出

**验收标准**:
- [ ] 证书符合 X.509 标准
- [ ] 私钥不落地存储
- [ ] 证书自动过期机制正常
- [ ] 支持证书文件导出

---

#### F-004: VPN 服务集成

**描述**: 通过 Android VpnService 捕获设备流量

**功能要求**:
- 创建 TUN 设备
- 配置路由捕获所有流量
- 解析 IP/TCP/UDP 数据包
- 转发流量到代理服务器

**验收标准**:
- [ ] 支持 Android 7.0+
- [ ] VPN 连接建立时间 < 3s
- [ ] 支持后台运行
- [ ] 前台服务通知正常显示

---

## 八、数据模型

### 8.1 TrafficRecord

```dart
class TrafficRecord {
  final String id;               // 唯一标识
  final DateTime timestamp;       // 时间戳
  final String method;           // HTTP 方法
  final String url;              // 完整 URL
  final String host;             // 主机名
  final String path;             // 路径
  final int statusCode;          // 状态码
  final int requestSize;         // 请求大小
  final int responseSize;        // 响应大小
  final int durationMs;          // 耗时 (毫秒)
  final String? requestType;     // 请求 Content-Type
  final String? responseType;    // 响应 Content-Type
  final bool isHttps;            // 是否 HTTPS
  final String? clientIp;        // 客户端 IP
}
```
